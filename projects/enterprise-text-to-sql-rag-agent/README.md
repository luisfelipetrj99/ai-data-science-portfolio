Enterprise-grade Text-to-SQL AI assistant capable of translating natural language business questions into SQL queries using Retrieval-Augmented Generation (RAG), semantic retrieval, and Large Language Models.

---

## Overview

This project implements a corporate AI analytics assistant designed to help business users query structured databases using natural language.

The system combines:
- LLM-based SQL generation
- RAG pipelines
- Semantic search
- FAISS vector retrieval
- Metadata-aware schema selection
- Prompt engineering
- Token optimization

The project compares multiple approaches:
1. Baseline full-context prompting
2. Retrieval-Augmented Generation (RAG)
3. Key-value metadata retrieval

---

## Business Problem

Enterprise databases often contain hundreds of tables and thousands of columns, making SQL querying difficult for non-technical users.

This project aims to:
- Reduce the technical barrier for analytics
- Improve self-service BI
- Optimize schema retrieval
- Reduce hallucinations in SQL generation
- Lower prompt/token costs
